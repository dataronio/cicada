# cicada

A dependently typed language
- with fulfilling type system
- based on game semantics
- for interactive theorem proving
- and practical verification tasks

## Contributing

- Compile: `sbt compile`
- Test: `./dev t`
- Install: `sbt publishLocal`
- Dependency: `"xieyuheng" %% "cicada" % "0.0.1"`
